  • git clone path-to-repository (путь к репозиторию)
  • git status (статус текущего репозитория)
  • git add path-to-file(добавить новый файл под контроль гита)
  • git commit -am "added new file index.html" (Зафиксировать изменения и дать им комментарий)
  • git push(Отправить все закомиченные изменнения в репозиторий на сервер)
  • git pull (Забрать все изменения)
